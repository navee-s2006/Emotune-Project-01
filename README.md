# 🎵 Emotune - Emotion Based Music Recommendation System

## 📌 Overview

Emotune is a web-based application that detects a user's facial emotion using the device camera and recommends songs based on the detected emotion. The project aims to enhance the music listening experience by automatically suggesting songs that match the user's mood.

---

## ✨ Features

- 😊 Real-time facial emotion detection
- 📷 Camera access using WebRTC
- 🎶 Emotion-based music recommendations
- 👤 User Registration and Login
- 📱 Responsive user interface
- ⚡ Fast and easy to use

## 🚀 How It Works

1. User registers or logs in.
2. The application requests camera permission.
3. The camera captures the user's face.
4. Face API detects the user's emotion.
5. Based on the detected emotion, suitable songs are recommended.
6. The user can play and enjoy the recommended music.
